# Examples

- `minimal.orgf` is a compact ORGF 1.0.0 model without an embedded reference.
- `minimal-self-contained.orgf` is the same kind of small model with the complete canonical ORGF 1.0.0 reference embedded immediately after `orgformat`.

Both files are MIT-licensed. The second file demonstrates the self-contained pattern for
sharing a model with a person, tool, or AI workflow that may not have the reference nearby.
