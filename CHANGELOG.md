# Changelog

## 1.0.0 — 2026-06-24

First public release of ORGF.

- Published the canonical ORGF 1.0.0 reference.
- Increased `model.desc` limit to 1000 characters; `block.desc` remains 500.
- Defined the official GitHub Releases registry for canonical embedded references.
- Simplified `hidden` semantics: model state, authoring rule, and normalization rule are separated from viewer/UI behaviour.
- Published a reference Node.js validator and public CC0 DemoPacks.
